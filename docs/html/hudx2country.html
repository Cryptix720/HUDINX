NAME

hudx2country - Lookup Country by IP Address.  

SYNTAX

hudx2country ip [ip2] [ip3] ...  

DESCRIPTION

Lookup Country by IP Address.  

EXAMPLES

$ hudx2country www.honeypot.co.uk

SE, Sweden

$ hudx2country 66.35.250.150

Sweden
